Glossary
========

.. glossary::
   :sorted:

   WSGI
      Web Server Gateway Interface. It is a specification about how web servers,
      like e.g. Apache with mod_wsgi, communicate with web applications. 
      It is a Python standard, described in detail in PEP 333.

   flask
      A micro framework used by eNMS.

   jinja
      A templating engine used by eNMS.

   sqlalchemy
      An SQL database abstraction library used by eNMS.

   sqlite
      An easy-to-use SQL database used by eNMS.

   werkzeug
      A WSGI library used by eNMS.
